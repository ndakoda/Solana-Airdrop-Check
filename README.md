

  Airdrop Scanner

A lightweight, open-source tool to check Eth wallet eligibility for past and active airdrops. Built with vanilla JavaScript and designed for transparency.

 Features

- Check wallet eligibility for 10+ Eth projects
- Real-time validation of Eth addresses
- Clean, responsive UI
- No data storage – all processing is client-side
- Open source and auditable

 Supported Projects
Uniswap (UNI)
Arbitrum (ARB)
Optimism (OP)
ENS (ENS)
Merkle (MKL)
dYdX (DYDX)
Injective (INJ)

- And more...


 Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Enter a Eth wallet address
4. Click "Check Eligibility"
